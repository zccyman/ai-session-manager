# Proposal: Browser Tab Export

## 概述

在 AI Session Manager 中添加浏览器标签页内容提取和管理功能。

## 问题

用户需要一种方式来：
1. 从浏览器标签页提取 Markdown 内容
2. 在统一界面中管理和搜索这些内容
3. 保存、复制和导出

## 解决方案

### 方案：浏览器扩展 + Web 应用

1. **浏览器扩展**：从网页提取对话内容，发送到 Web 应用
2. **Web 应用后端**：存储和管理标签页内容
3. **Web 应用前端**：展示、搜索、保存、复制功能

## 功能需求

### 浏览器扩展
- 提取当前标签页的对话内容
- 提取所有标签页的对话内容（批量）
- 发送到 AI Session Manager 后端
- 一键发送所有提取的内容
- 显示提取状态和进度

### Web 应用后端
- 标签页内容 CRUD API
- 搜索 API
- 导出 API（单个 Markdown / 批量导出到目录）

### Web 应用前端
- 标签页内容列表
- 内容详情查看
- 搜索功能
- 复制/下载功能
- 视图切换（会话/标签页）
- 批量导出到指定目录（默认 `G:\knowledge\source\browser-export`，支持手动修改）

## 目标用户

使用 AI 聊天应用（如 TabBitBrowser）并希望保存和管理对话内容的开发者。

## 成功标准

- [x] 浏览器扩展能提取并发送内容
- [x] 浏览器扩展能批量提取所有标签页
- [x] 后端能存储和检索内容
- [x] 前端能展示、搜索、复制内容
- [x] 支持批量导出到指定目录，可自定义路径
